## 📌 Project Overview

BattleMode is a **gamified web application** built using Django that helps students stay consistent, improve productivity, and prepare for placements.

Users complete real-life tasks (missions), submit proof, and earn XP only after admin approval — making it a **real accountability system**, not just a fake tracker.

## ⚙️ How to Run Locally

```bash
git clone https://github.com/manas2801/Battlemode.git
cd BattleMode

env\Scripts\activate

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
```
